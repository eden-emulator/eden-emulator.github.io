Nintendo Switch Emulator

Eden is an experimental open-source emulator for the Nintendo Switch, built with performance and stability in mind.

It is written in C++ with cross-platform support for Windows, Linux, MacOS, FreeBSD, Solaris and Android.
Support for MacOS, FreeBSD and Solaris is W.I.P.
