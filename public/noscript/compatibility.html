System Requirements

CPU

Any amd64 CPU with support for the FMA instruction set, or an ARM64-v8a or newer CPU. 6 threads or more are recommended.

Minimum Intel Core i5-4430 / AMD Ryzen 3 1200
Recommended Intel Core i5-11400 / AMD Ryzen 5 3600 / Snapdragon X Plus
Optimal Intel Core i9-13900K / AMD Ryzen 7 7800X3D / Snapdragon X Elite

SoC (Android)

Using a cooler is always recommended. Similarly-specced SoCs should usually work.

Minimum Snapdragon 865 / Tensor G1 / Exynos 2200 / Dimensity 8200
Recommended Snapdragon 8 Gen 1 / Tensor G4 / Exynos 2400 / Dimensity 8400
Optimal Snapdragon 8 Gen 3 / Snapdragon G3 Gen 3 / Tensor G5 / Exynos 2400-12 / Dimensity 9400

Laptop CPU

Mobile and Laptop CPUs generally need to be far newer to effectively run Eden due to their thermal throttling and lower power draw.

Recommended Intel Core i5-13420H / AMD Ryzen 7 5700U
Optimal Intel Core Ultra 7 265H / AMD Ryzen AI 9 HX 370

Dedicated Graphics

OpenGL 4.6 or Vulkan 1.1 compatible hardware and drivers are mandatory. Half-float support and VP9 decoding support are recommended.

Minimum for Linux NVIDIA GeForce GT 1030 4GB / AMD Radeon R7 240 4GB
Minimum for Windows NVIDIA GeForce GTX 1050 4GB / AMD Radeon RX 550 4GB
Recommended NVIDIA GeForce GTX 1660 6GB / AMD Radeon RX 5500 8GB (with “Force maximum clocks” enabled)
Optimal NVIDIA Geforce RTX 3060 12GB / AMD Radeon RX 6700 10GB (with “Force maximum clocks” enabled)

Integrated Graphics

Integrated graphics will produce very low performance. A dedicated GPU will produce better results on all scenarios. This is only for listing iGPU support.

Minimum For Windows Intel UHD Graphics 730 / AMD Radeon Vega 3
Minimum For Linux Intel HD Graphics 4400 / AMD Radeon R5 Graphics
Recommended Intel Iris Xe Graphics / AMD Radeon 680M
Recommended Latest Intel Graphics / AMD Radeon 8060S

Android

Must support Vulkan 1.1+. Note that most PowerVR and Xclipse GPUs are unsupported except for the very newest.

Minimum Recommended Qualcomm Adreno 650 (with Legacy build) / ARM Mali G78
Recommended Qualcomm Adreno 740 / ARM Mali G715
Optimal Qualcomm Adreno A32/750 / ARM Immortalis G925 / Xclipse 950 / PowerVR DXT-48

RAM

Since an integrated GPU uses system RAM as its video memory (VRAM), our memory requirement in this configuration is higher. Highest possible frequency is strongly recommended.

Dedicated GPU Desktop

Minimum 8GB
Recommended 16GB
Optimal 32GB

Android

Minimum 8GB

Laptop/iGPU Desktop

Minimum 12GB
Recommended 16GB
Optimal 32GB
